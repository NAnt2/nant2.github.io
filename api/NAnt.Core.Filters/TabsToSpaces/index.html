Syntax

[ElementName("tabstospaces")]
public class TabsToSpaces : Filter, IDisposable

Examples

Replace all tabs with four spaces.

<tabtospaces tablength="4" />

Remarks

The NAnt.Core.Filters.TabsToSpaces filter replaces tabs in a text file with spaces.

Filters are intended to be used as a element of a NAnt.Core.Filters.FilterChain.

Promotes versioned files to the next promotion group.

Syntax

[TaskName("pvcspromotegroup")]
public sealed class PVCSPromoteGroupTask : PVCSSingleEntityTask

Examples

Promotes all files in the root of the project database specified by the project-database property. The files are promoted from the DEV promotion group to the next. Promotion will not take place across branches. 

<pvcspromotegroup projectdatabase="${project-database}" promotiongroup="DEV" entity="/" acrossbranches="false"/>

Promotes all files in the project database specified by the project-database property. The files are promoted from the SYSTEST promotion group to the next. Promotion will take place across branches. 

<pvcspromotegroup projectdatabase="${project-database}" promotiongroup="SYSTEST" entity="/" includesubprojects="true" acrossbranches="true"/>

Remarks

This task uses the promotegroup PCLI command to promote versioned files.

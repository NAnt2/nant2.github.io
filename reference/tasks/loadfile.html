Nested elements

<filterchain>

The filterchain definition to use.

Represent a chain of NAnt filters that can be applied to a NAnt.Core.Task.

A FilterChain represents a collection of one or more filters that can be appled to a NAnt.Core.Task such as the NAnt.Core.Tasks.CopyTask. In the case of the NAnt.Core.Tasks.CopyTask, the contents of the copied files are filtered through each filter specified in the filter chain. Filtering occurs in the order the filters are specified with filtered output of one filter feeding into another.

:--------:--->:----------:--->:----------: ... :----------:--->:--------:
:.Source.:--->:.Filter 1.:--->:.Filter 2.: ... :.Filter n.:--->:.target.:
:--------:--->:----------:--->:----------: ... :----------:--->:--------:

A list of all filters that come with NAnt is available here.

The following tasks support filtering with a FilterChain:

  • NAnt.Core.Tasks.CopyTask
  • NAnt.Core.Tasks.MoveTask

Examples

Load file message.txt into property "message". 

<loadfile
file="message.txt"
property="message" />

Load a file using the "latin-1" encoding. 

<loadfile
file="loadfile.xml"
property="encoded-file"
encoding="iso-8859-1" />

Load a file, replacing all @NOW@ tokens with the current date/time. 

    <loadfile file="token.txt" property="token-file">
    <filterchain>
        <replacetokens>
            <token key="NOW" value="${datetime::now()}" />
        </replacetokens>
    </filterchain>
</loadfile>

Remarks

Unless an encoding is specified, the encoding associated with the system's current ANSI code page is used.

An UTF-8, little-endian Unicode, and big-endian Unicode encoded text file is automatically recognized, if the file starts with the appropriate byte order marks.
